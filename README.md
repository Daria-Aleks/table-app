# Выполнены основные цели: 
- реализована таблица с помощью запроса, добавлен input для поиска информации по таблице, ширина таблицы 100%, но не более 1200px, использованы только FetchAPI для работы с HTTP-запросами.

## Выполнены дополнительные цели:
- Добавлена возможность сортировки элементов таблицы для колонок с ФИО, возрастом, полом и адресом. Сортировка должна иметь три состояния: по возрастанию, по убыванию, без сортировки. 
- Добавлена возможность изменения ширины каждой колонки таблицы. Минимальная ширина столбца при такой возможность должна быть равна 50px.
- Реализована обработку возможных ошибок при работе с HTTP-запросами.
- Добавлен компонент для отображения модального окна с подробной информацией о пользователе по клику на строку таблицы. В окне выводится следующая информация: ФИО, возраст, адрес (город и название улицы), рост, вес, номер телефона и email-адрес.


In the project directory, you can run:

### `npm start`
